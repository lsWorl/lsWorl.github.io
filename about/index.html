关于我

你好，我是 LsWorld。这里是我的个人知识库与技术博客，主要用来记录机器学习、代码实践、计算机基础和工程踩坑。

我希望这个博客不只是文章归档，也能成为一个持续生长的学习地图：把学过的概念讲清楚，把写过的代码留下来，把踩过的坑整理成以后可以复用的经验。

当前关注方向：机器学习、深度学习、Python、前端工程、后端开发、嵌入式与计算机基础。

博客内容

机器学习与深度学习

记录监督学习、分类算法、回归模型、模型评估、数学推导和实验过程。相比只堆公式，我更希望把算法背后的直觉、适用场景和代码实现一起说明白。

编程与工程实践

这里会整理 JavaScript、TypeScript、Vue、Node.js、NestJS、Python、数据库、部署、工程化等内容。很多文章来自真实学习和项目过程中的问题复盘。

计算机基础

包括数据结构、操作系统、计算机网络、计算机组成原理等基础知识。基础内容看起来慢，但它们会在很多具体问题里反复出现。

嵌入式与硬件相关

记录 STM32、FreeRTOS、传感器模块、串口协议、外设驱动等学习笔记。偏实践，也会尽量保留关键代码和调试思路。

技术栈

  • 语言：Python、JavaScript、TypeScript、C
  • 前端：Vue、Vue Router、Vuex、Webpack（忘的差不多了…）
  • 后端：Node.js、NestJS、TypeORM、MySQL
  • 机器学习：NumPy、Pandas、Scikit-learn、基础数学推导（目前主要方向）
  • 嵌入式：STM32、FreeRTOS、常见传感器与通信协议
  • 博客：Hexo、Butterfly、GitHub Pages

我在整理的学习路线

  1. 夯实数学、数据结构、操作系统、网络、组成原理等基础。
  2. 用 Python 复现机器学习经典算法，理解模型如何从数据中学习。
  3. 通过前后端项目练习工程能力，把知识落到可运行的应用里。
  4. 持续记录问题、复盘方法，并把零散笔记整理成体系化文章。

关于这个博客

这个站点使用 Hexo + Butterfly 搭建，部署在 GitHub Pages。博客里的文章大多是学习过程中的阶段性总结，可能会随着理解加深不断更新。

如果某篇文章帮你解决了问题，那它就完成了自己的使命；如果你发现有不准确的地方，也欢迎通过 GitHub 与我交流。
